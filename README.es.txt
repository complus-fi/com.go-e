Carga de forma simple e inteligente. Carga tu coche eléctrico de manera económica e inteligente.

Compatibilidad con los modelos Home+(V3), Gemini, CORE y PRO de cargadores go-e, tanto con acceso por red local como con acceso en la nube.

Para el acceso local, activa la API V2 en los ajustes de la aplicación móvil de go-e.

Para el acceso en la nube, activa la Cloud API en los ajustes de la aplicación móvil de go-e e introduce el número de serie de tu dispositivo y la clave de Cloud API generada en los ajustes del dispositivo Homey.

Esta aplicación Homey está desarrollada de forma independiente por Complus Ky con el apoyo de go-e GmbH. Para soporte y consultas, ponte en contacto directamente con el desarrollador.
