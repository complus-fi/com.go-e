Lad enkelt og smart. Lad elbilen din til lav kostnad og på en intelligent måte.

Støtte for Home+(V3)-, Gemini-, CORE- og PRO-modellene av go-e-ladere med både lokal nettverkstilgang og skybasert tilgang.

For lokal tilgang aktiverer du V2 API i innstillingene i go-e-mobilappen.

For skybasert tilgang aktiverer du Cloud API i innstillingene i go-e-mobilappen og legger inn enhetens serienummer og generert Cloud API-nøkkel i enhetsinnstillingene i Homey.

Denne Homey-appen er utviklet uavhengig av Complus Ky med støtte fra go-e GmbH. For support og spørsmål, vennligst kontakt utvikleren direkte.
