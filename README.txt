Simply Charge Smart. Charge your electric car at a low cost and intelligently.

Support for the Home+(V3), Gemini, CORE and PRO models of go-e chargers both with local network and cloud based access.

For local access, enable the V2 API in the go-e mobile app settings.

For cloud based access, enable the Cloud API in the go-e mobile app settings and provide your device's serial number and generated Cloud API key in the Homey device settings.

This Homey app is developed independently by Complus Ky with support from go-e GmbH. For support and inquiries, please contact the developer directly.
