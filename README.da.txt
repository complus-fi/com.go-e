Oplad enkelt og smart. Oplad din elbil billigt og intelligent.

Understøttelse af Home+(V3), Gemini, CORE og PRO modellerne af go-e opladere med både lokal netværksadgang og cloudbaseret adgang.

For lokal adgang skal du aktivere V2 API i indstillingerne i go-e mobilappen.

For cloudbaseret adgang skal du aktivere Cloud API i indstillingerne i go-e mobilappen og angive enhedens serienummer samt den genererede Cloud API-nøgle i Homey-enhedsindstillingerne.

Denne Homey-app er udviklet uafhængigt af Complus Ky med støtte fra go-e GmbH. Kontakt venligst udvikleren direkte for support og forespørgsler.
