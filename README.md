# com.go-e

Simply Charge Smart. Charge your electric car at a low cost and intelligently.

Support for the Home+ (V3), Gemini, CORE, and PRO models of go-eChargers with V2 API.
Enable the V2 API in the go-eCharger mobile app settings.
Enable the Cloud API and use the serial number and generated Cloud API Key to use with Cloud devices.

This Homey app is developed independently by Complus Ky and is not an official product of go-e GmbH.
For support and inquiries, please contact the developer directly.
