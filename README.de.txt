Einfach intelligent laden. Lade dein Elektroauto kostengünstig und intelligent.

Unterstützung für die Home+(V3)-, Gemini-, CORE- und PRO-Modelle der go-e Charger mit lokalem Netzwerkzugriff und cloudbasiertem Zugriff.

Für den lokalen Zugriff aktiviere die V2 API in den Einstellungen der go-e Mobile-App.

Für cloudbasierten Zugriff aktiviere die Cloud API in den Einstellungen der go-e Mobile-App und trage die Seriennummer deines Geräts sowie den generierten Cloud API-Schlüssel in den Homey-Geräteeinstellungen ein.
