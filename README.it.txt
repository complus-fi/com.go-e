Ricarica in modo semplice e intelligente. Ricarica la tua auto elettrica a basso costo e in modo intelligente.

Supporto per i modelli Home+(V3), Gemini, CORE e PRO dei caricabatterie go-e, sia con accesso in rete locale sia con accesso cloud.

Per l'accesso locale, abilita la V2 API nelle impostazioni dell'app mobile go-e.

Per l'accesso cloud, abilita la Cloud API nelle impostazioni dell'app mobile go-e e inserisci il numero di serie del dispositivo e la chiave Cloud API generata nelle impostazioni del dispositivo Homey.

Questa app Homey è sviluppata in modo indipendente da Complus Ky con il supporto di go-e GmbH. Per assistenza e richieste, contatta direttamente lo sviluppatore.
