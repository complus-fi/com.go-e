Заряжайте просто и умно. Заряжайте свой электромобиль недорого и интеллектуально.

Поддерживаются модели зарядных устройств go-e Home+(V3), Gemini, CORE и PRO как с доступом по локальной сети, так и через облако.

Для локального доступа включите V2 API в настройках мобильного приложения go-e.

Для доступа через облако включите Cloud API в настройках мобильного приложения go-e и укажите серийный номер устройства и сгенерированный ключ Cloud API в настройках устройства Homey.

Это приложение Homey независимо разработано компанией Complus Ky при поддержке go-e GmbH. По вопросам поддержки и сотрудничества обращайтесь напрямую к разработчику.
